# Ryohei Nagasawa — Portfolio

A 3D interactive portfolio website built with Three.js, vanilla JavaScript, and a minimal Node.js/Express server.

## Quick Start

```bash
cd portfolio
npm install
npm start
```

Then open **http://localhost:3000** in your browser.

## Features

- **3D Neural Network** — Live particle system with connection lines and mouse parallax
- **Floating Wireframe Geometries** — Icosahedra, octahedra, and tetrahedra drifting in 3D space
- **Scroll Reveal Animations** — Sections animate in as you scroll
- **Glassmorphism Nav** — Navigation blurs and darkens on scroll
- **Fully Responsive** — Works on desktop and mobile

## Stack

- Three.js (CDN) — 3D rendering
- Express.js — Static file server
- Vanilla JS — Interactions & animations
- Google Fonts (Inter + JetBrains Mono)

## Requirements

- Node.js 16+ (any modern version works)
- npm
